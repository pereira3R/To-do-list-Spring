# To-do-list Spring Boot

Activity carried out by the Infocorp back-end team - completed in November 2023
- First Simple Project with Spring Boot

## Parts of the project

- Controllers
  - UserController
  - TaskController
- Models
  - User
  - Task
- Repositories
  - UserRepositorie
  - TaskRepositorie
- Services
  - UserService
  - TaskService
- Exceptions
  - ErrorResponse
  - GlobalExceptionHandler
- Configs
  - SwaggerConfig
  - WebConfig
- View
  - index.html
  - script.js
  - css.style (+ Boostrap 5)
